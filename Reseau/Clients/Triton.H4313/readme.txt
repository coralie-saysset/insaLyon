Pour construire l'exécutable :

 * Ouvrir src/VideoSurveillance.sln avec Microsoft Visual Studio 2005

 * Dans le menu "Build", cliquer sur "Build Solution"

   S'il y a des erreurs à la compilation, essayer de faire un "Clean Solution" avant de compiler.


NB: Une version déjà compilée se trouve dans le répertoire "bin".
